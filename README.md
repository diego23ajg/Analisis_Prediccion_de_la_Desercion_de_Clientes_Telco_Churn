# Predicción de deserción de clientes (Churn) en telecomunicaciones
Predicción de deserción de clientes (Churn) en telecomunicaciones usando XGBoost y Random Forest. Análisis de datos, limpieza, preprocesamiento con pandas, scikit-learn y SHAP. Precisión >80%. [Telco Customer Churn - Kaggle dataset]. Código en Python.
